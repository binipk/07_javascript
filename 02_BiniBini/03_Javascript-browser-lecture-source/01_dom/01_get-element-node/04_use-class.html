✅ class명을 이용한 요소 선택

📌 getElementsByClassName("클래스명")
👉 해당 class를 가진 모든 요소들을 HTMLCollection으로 반환
👉 같은 class명이 여러 요소에 쓰여도 모두 선택 가능 (중복 허용)
👉 반환된 HTMLCollection은 배열처럼 생겼지만 forEach 안 됨
👉 배열로 바꾸려면 [...컬렉션] 또는 Array.from(컬렉션) 사용
